1. title : web의 제목을 결정해 주는 역할, 검색엔진에 띄워줌
2. utf-8 : 글씨를 안깨지게 해주는 역할
*character set - 문자 붙이기 (utf-8로 문서를 읽어라)
3. head : 본문의 제목을 나타내는 태그
4. body : head를 설명해주는 태그
5. html : body와 head를 묶는 최 고위직 태그(hypertext markup language)
6. !doctype html : html 태그 위에 관용적으로 적는 태그
* 서열 : !doctype html=html > head=body > title=meta
